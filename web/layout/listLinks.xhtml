ListLinks and navLink

b:listLinks is a container for NavLinks. this can be very useful to create Sidebars with links.

NavLinks are important, in fact they are used also in

As you can see in the example below, inside b:listLinks you can have four kinds of b:navLink:

  • a link that points to an outcome
  • an external link
  • a header link
  • a separator link
The outcome and external links can have an icon (left or right, default is left).

You can control the icon position by specifying it in the attribute iconAlign
