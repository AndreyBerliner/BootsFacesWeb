BootsFaces: next-gen JSF Framework

Examples

These BootsFaces examples are ports of Twitter Bootstrap 3 examples.

They are built using JSF and BootsFaces's components, to show the framework in action and how it can be used to build your pages.

You can get them as a point of start for developing your projects.

Starter template

Nothing but the basics: compiled CSS and JavaScript along with a container.

Grids

Multiple examples of grid layouts with all four tiers, nesting, and more.

Jumbotron

Build around the jumbotron with a navbar and some basic grid columns.

Navbar

Super basic template that includes the navbar along with some additional content.

Static top navbar

Super basic template with a static top navbar along with some additional content.

Fixed navbar

Super basic template with a fixed top navbar along with some additional content.
