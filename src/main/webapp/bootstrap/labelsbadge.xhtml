Labels and Badges

Caveat: Bootstrap 4 is going to drop the labels and badges of Bootstrap 3. So there's a certain risk this component won't be supported after migrating BootsFaces to Bootstrap 4.

Note: starting from Bootstrap 3, the Badge has no more severity colors and some severity level names for label have changed(important => danger and inverse=> primary).
