ScrollUp (<b:scrollUp />)

<b:scrollUp> create a customisable 'Scroll to top' feature that will work with any website.

Basic usage

By default, scrollup is put on the bottom right corner without images or text. It is basically a link to scroll up With attributes you can setup link text or distance between display the item. But is using a custom css that you can really customize this component. The reference to the css id selector is done with the "name" attribute, as displayed in the example
