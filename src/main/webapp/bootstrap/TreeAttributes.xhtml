Attributes of <b:tree >
Attribute Default value Description
id (none) Unique identifier of the component in a namingContainer.
rendered false Boolean value to specify the rendering of the component, when set to false the component will not be rendered.
value (none)
List (none) Tree model value. Requires a java.util.List<net.bootsfaces.component.tree.model.Node> to work.
nodeSelectionListener (none) Selection listener called on selection changed.
showTags false Boolean value to specify if tags was show or not.
showIcon false Boolean value to specify if icons are rendered or not.
showCheckbox false Boolean value to specify if checkbox is rendered or not.
enableLinks false Boolean value to specify if enable href link.
collapseIcon (none) Glyphicons of collapse.
expandIcon (none) Glyphicons of expand.
color (none) Text color.
update (none) Component(s) to be updated with ajax.
