ScrollSpy (<b:scrollSpy />)

<b:scrollSpy> is a wrapper of the native bootstrap scrollspy function. it can be used to provide automatic menu highlight during scrolling. The addition of bootsfaces is the possibility to define an action listener in the backend JSF bean that receive via ajax the name of current highlighted section.
Is very important that, the section that you use as a scroll container, has the css position property set to "relative".
