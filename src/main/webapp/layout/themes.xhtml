Themes

BootsFaces comes with the Bootstrap default theme and 16 Bootswatch themes. Plus, you can use an arbitrary third-party or custom Bootstrap theme.

Theme CSS support

The BootsFaces_USETHEME context-param in your web.xml controls the loading of the bootstrap-theme.css

By default this parameter is set to false. You need to explicitely set it to true or default to have the default bootstrap-theme.css loaded.
