RTL support

BootsFaces adds some limited support for RTL (right-to-left) languages. However, RTL isn't fully supported by Bootstrap itself. However, you can add RTL by adding this library:

https://github.com/morteza/bootstrap-rtl

Basic usage

Many BootsFaces components have an dir attribute. Set it to dir="rtl" to activate RTL rendering.
